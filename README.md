# コリドール (Quoridor) for GAS

Google Apps Script (GAS) 上で動作する、2人対戦用のボードゲーム「コリドール」です。相手よりも先に反対側のゴールラインへ到達することを目指します。自分の手番では「コマを動かす」か「カベを置いて相手を邪魔する」かの戦略的な選択が求められます。

## 特徴

* **選べる難易度**: 7x7（かんたん）、9x9（ふつう）、11x11（むずかしい）の3種類のボードサイズを選択可能です。
* **直感的な操作モード**: 「コマを動かすモード」と「カベを置くモード」をボタンで切り替えて操作します。
* **動的ハイライト**: 移動可能なマスや、カベを設置可能な場所がリアルタイムで強調表示されます。
* **スマートなルール判定**:
    * カベのクロス設置や接続設置の禁止。
    * ゴールまでの道を完全にふさぐようなカベ設置の制限。
    * 相手のコマを飛び越える「ジャンプ」ルールの実装。
* **レスポンシブデザイン**: Google Fonts（M PLUS Rounded 1c）を使用した、親しみやすく見やすいインターフェースを採用しています。

## 技術構成

* **サーバーサイド**: Google Apps Script (`code.gs`)
* **クライアントサイド**: HTML5, CSS3, JavaScript (`index.html` 内に統合)
    * **UI**: CSS Grid を利用した動的なボード描画。
    * **ロジック**: BFS（幅優先探索）アルゴリズムを用いた、ゴール到達可能性の判定（カベ設置制限用）。

## ファイル構成

* `code.gs`: Webアプリケーションとして公開するための doGet 関数が含まれています。ブラウザのタブタイトルの設定などもここで行います。
* `index.html`: ゲームの構造、スタイル、およびすべてのフロントエンド・ロジックが含まれています。

## 遊び方

1.  **セットアップ**: ボードの大きさを選択して「ゲーム開始」をクリックします。
2.  **移動**: 「コマを動かす」モードで、ハイライトされた隣接するマス（または相手を飛び越えるマス）をクリックして進みます。
3.  **妨害**: 「カベを置く」モードで、カベを置きたいスロットをクリックします。カベはコマ2個分の長さがあります。
    * 注意：相手や自分のゴールへの道を完全に無くすようなカベは置けません。
4.  **勝利条件**: 相手よりも先に、ボードの反対側の端（プレイヤー1は右端、プレイヤー2は左端）に到達したプレイヤーの勝利です。

## デプロイ方法

1.  Google Apps Script プロジェクトを作成します。
2.  `code.gs` の内容をスクリプトエディタに貼り付けます。
3.  `index.html` という名前でHTMLファイルを作成し、その内容を貼り付けます。
4.  「新しいデプロイ」から「ウェブアプリ」を選択し、アクセス権限を設定して公開します。
